3.讨论结果 

（1）需要从已有个人经验中和对话数据中，穷举多样的应用场景

（2）需要明确系统的主要功能场景：
信息查询（查询号码、号码反查、来电查询），电话转接、信息增改
信息查询分为三类：一是通过名字、职务、单位查电话号码；二是通过电话号码查询姓名、职务、单位；三是询问总机什么时间有人接过他
